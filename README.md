# FUTURE_DS_01
📊 Excited to share my Super Store Sales Dashboard built using Microsoft Power BI!

This project analyzes Super Store sales data to track business performance, profit trends, customer segments, shipping methods, and sales forecasts. The dashboard helps identify key insights for better business decision-making.

🔹 Created interactive KPI cards to monitor Total Sales, Total Profit, and Average Delivery.
🔹 Built dynamic visualizations including bar charts, line charts, pie/donut charts, and sales forecasting.
🔹 Used Power Query for data cleaning and transformation.
🔹 Applied DAX measures for calculations and interactive reporting.
🔹 Added filters and slicers for better user experience and data exploration.

📈 Key Insights:
• Standard Class generated the highest sales.
• Office Supplies was the top-performing product category.
• California recorded the highest sales among all states.
• Consumer Segment contributed the largest share of total sales.
• Monthly sales and profit trends helped identify seasonal business patterns.
• Built a 15-day sales forecast to support future planning.

This project strengthened my skills in Power BI, Power Query, DAX, Data Modeling, Data Cleaning, Data Visualization, and Business Intelligence.
